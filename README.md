# Mosaic Table Game

**Mosaic Table** – это интерактивная игра‑головоломка, где игроку нужно находить элементы в заданном порядке. Таблица может быть большой, а элементы нумеруются цифрами. Порядок поиска задаётся опциями **прямой** (от 1 к N) или **обратный** (от N к 1).

---

## 🎮 Геймплей

1. **Выбор режима** – при запуске можно выбрать один из режимов отображения элементов, которые нужно отыскивать:
   - **Стандартный** – все элементы отображаются и нумеруются.
   - **Память** – номера элементов исчезают через **10 секунд**, после чего игроку нужно в правильном порядке указать нужные ячейки.
2. **Последовательный поиск** – игрок последовательно кликает по ячейкам в требуемом порядке. При ошибке игра подсвечивает неверный выбор.
3. **Режимы порядка** –
   - **Прямой** – искать от 1 к N.
   - **Обратный** – искать от N к 1.

---

## 📈 Зачем это нужно?
- **Развитие усидчивости**: игра требует длительного внимания, особенно на больших таблицах.
- **Тренировка внимательности**: поиск нужного элемента среди множества визуальных предметов.
- **Развитие памяти**: режим «Память» заставляет запоминать позиции элементов после исчезновения их номеров.

## 🛠 Технологии
- **HTML5 / CSS3** – разметка и стилизация.
- **JavaScript (ES6)** – логика игры.
- **D3.js v7** – используется для рендеринга мозаики и анимаций.
- **Google Fonts** – шрифты `Outfit` и `Manrope`.

## 🎨 Настройка внешнего вида
В начале `index.html` объявлены CSS‑переменные, которыми можно менять цветовую схему и размеры:
```css
:root {
    --bg-color: #0f172a;      /* фон */
    --text-color: #e2e8f0;    /* текст */
    --primary-color: #3b82f6; /* основной */
    --primary-hover: #2563eb; /* наведение */
    --accent-color: #8b5cf6;  /* акцент */
}
```
Отредактируйте их под ваш бренд.
