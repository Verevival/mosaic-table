# Mosaic Table

![Mosaic Table preview](https://raw.githubusercontent.com/Verevival/mosaic-table/main/index.html#screenshot)

## 📊 О проекте
`Mosaic Table` — интерактивная таблица‑мозаика, созданная на базе **D3.js**. Она позволяет визуализировать любые табличные данные в виде цветных мозаичных ячеек, поддерживая:

- **Динамические** переходы при изменении данных;
- **Настраиваемые** цветовые схемы (primary, accent и пр.);
- **Поддержку** мобильных устройств (адаптивный макет);
- **Интерактивные** подсказки и анимацию при наведении.

## ✨ Основные возможности
- **Отображение** произвольных CSV/JSON‑данных в виде мозаики;
- **Плавные** анимации при изменении размеров/цветов ячеек;
- **Пользовательские** стили через CSS‑переменные;
- **Полностью** клиент‑сайд – без серверных зависимостей.

## 🚀 Как запустить
1. Откройте в браузере публичный URL:
   ```
   https://verevival.github.io/mosaic-table/
   ```
2. Если хотите локально протестировать, склонируйте репозиторий и откройте `index.html` в браузере:
   ```bash
   git clone https://github.com/Verevival/mosaic-table.git
   cd mosaic-table
   open index.html   # или двойным кликом по файлу
   ```

## 🛠 Технологии
- **HTML5 / CSS3** – разметка и стилизация;
- **JavaScript (ES6)** – логика интерактивности;
- **D3.js v7** – построение и анимация визуализаций;
- **Google Fonts** – шрифты `Outfit` и `Manrope`.

## 🎨 Настройка стилей
В файле `index.html` в секции `<style>` объявлены CSS‑переменные, которые легко менять:
```css
:root {
    --bg-color: #0f172a;
    --text-color: #e2e8f0;
    --primary-color: #3b82f6;
    --primary-hover: #2563eb;
    --accent-color: #8b5cf6;
}
```
Подправьте значения под свой бренд или цветовую схему.
